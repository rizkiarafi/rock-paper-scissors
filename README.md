# Rock Paper Scissors Game (UI Version)

A simple **Rock–Paper–Scissors** game built with **HTML, CSS, and JavaScript**.  
This version includes a **user interface** — no more `prompt()` and console-only gameplay.

Players interact using buttons, and the game displays results directly on the screen.

---

## Features

- Button-based input (Rock / Paper / Scissors)
- Bot with random choices
- Live score display
- Round counter
- Result messages (Win / Lose / Draw)
- Match result after 5 rounds
- Clean UI layout

---

## Game Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choice = Draw

---

## How It Works

1. Player clicks a button:
   - Rock
   - Paper
   - Scissors
2. Bot randomly selects a move
3. Game logic compares choices
4. UI updates:
   - Scores
   - Round number
   - Result message
5. After 5 rounds, final winner is displayed

---

## How to Run

1. Open `index.html` in a browser
2. Click one of the buttons to play
3. Watch the score and results update in real-time
4. Game ends automatically after 5 rounds
